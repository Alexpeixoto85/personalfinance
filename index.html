Sincronização (GitHub Gist)

Sincronize seus dados entre celular, tablet e computador usando um Gist privado do GitHub. Use o botão abaixo para criar o token já com a permissão gist ativada, copie o código gerado (ghp_...) e cole aqui.

Criar token no GitHub (escopo gist)
  1. Toque no botão acima (faça login no GitHub se pedir).
  2. Role até o fim e clique em Generate token.
  3. Copie o código ghp_... e cole no campo abaixo.
  4. Clique em Enviar para criar seu Gist privado.
